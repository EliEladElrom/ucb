Install & set:

``> git clone https://github.com/eladelrom/ucb``
``> npm install``

Add '/.bash_profile' alias for cli shortcut;

``alias cli='node /[location]/ucb/bin/cli.js'`` 

Use 'cli' alias:

``>cli help``

``>cli daemon --start``

``>cli daemon --node``

``>cli block --create``

``>cli node --add``

``>cli node --get``
